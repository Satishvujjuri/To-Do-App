# 📝 To-Do App
A simple and interactive To-Do List web application built completely using JavaScript DOM manipulation.

## 🚀 Features
- Create tasks dynamically using JavaScript
- Add tasks using button click
- Add tasks by pressing Enter key
- Mark tasks as completed using checkbox
- Completed tasks get line-through effect
- Edit existing tasks
- Delete tasks
- Clean and simple user interface

## 🛠️ Technologies Used
- HTML
- JavaScript
- DOM Manipulation
- CSS Styling using JavaScript

## 📌 Project Highlights
This project focuses on practicing:
- Creating HTML elements dynamically with JavaScript
- Styling elements using JavaScript
- Handling user interactions with event listeners
- Working with input values
- Managing dynamic task creation and deletion
